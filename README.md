# NAME

nqp-configure

# DESCRIPTION

Provide support modules for [NQP](https://github.com/perl6/nqp) and [Rakudo](https://github.com/rakudo/rakudo)
`Configure.pl` scripts.

## Modules Provided

- `NQP::Config`
- `NQP::Macros`
